## 권한 보완 필요 항목

| 항목 | 관련 SRS | 보완 필요 내용 |
| --- | --- | --- |
| 설비 제어 명령 | SRS-009 | 제어 명령 실행 권한을 일반 사용자 내 세부 역할로 분리해야 한다. 예: 일반 조회자, 설비 제어자, 승인자. |
| 관리자 권한 변경 | SRS-004 | 권한 부여, 변경, 회수 권한을 가진 관리자 범위를 별도 정의해야 한다. |
| 파일 정책 | SRS-026 | 정책 적용 대상 범위가 사용자별, 그룹별, 전체인지 명시해야 한다. |
| 모델 버전 변경 | SRS-025, SRS-039 | 모델 변경 승인 절차와 롤백 권한을 정의해야 한다. |
| VectorDB 생성/공유 | SRS-016, SRS-032 | 사용자가 생성한 VectorDB의 소유자, 공유 범위, 삭제 권한을 정의해야 한다. |
