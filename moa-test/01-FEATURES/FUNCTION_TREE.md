# 기능 트리

기준 문서: `SRS.md`  
범위: `SRS-001` ~ `SRS-039`

## 1. MOA 시스템

### 1.1 사용자, 인증, 세션
- 회원가입: 신규 계정 생성, 필수 정보 검증 (`SRS-002`)
- 로그인: ID/PW 기반 인증, 자격 증명 검증 (`SRS-003`)
- 인증 및 세션 보안 정책: 로그인 실패 제한, 계정 잠금, 세션 만료, 관리자 권한 변경 감사 (`SRS-004`)
- 다중 클라이언트 접속 성능 보장 (`SRS-001`)

### 1.2 알람 및 선제 대응
- 알람 발생 알림: 설비 알람 감지, 화면/팝업 통지 (`SRS-006`)
- 알람 히스토리 조회: 최신순 조회 (`SRS-006`)
- 알람 처리 가이드 제공: 코드, 유형, 심각도 기반 가이드 조회 (`SRS-005`)
- 알람 이메일 발송: 담당자 이메일 통지, 발송 로그 기록, 대체 통지 (`SRS-007`)
- 알람 SMS 발송: 담당자 SMS 통지, 발송 결과 기록, 대체 통지 (`SRS-008`)

### 1.3 설비 제어 명령
- 자연어 명령 입력: 자연어를 제어 명령으로 변환 (`SRS-009`)
- 명령 권한 검증: 권한 있는 사용자만 명령 전송 (`SRS-009`)
- 실행 전 재확인 및 취소: 사용자 확인 후 전송 (`SRS-009`)
- 위험 명령 차단 및 감사 로그 (`SRS-009`)

### 1.4 근거 기반 질의 응답
- 이벤트 이력 조회: 자연어 조건 해석, DB 검색, 시간순 결과 제공 (`SRS-010`)
- 알람 원인 및 해결 방안 제공: 지식 베이스 검색, 표준 조치 정리 (`SRS-011`)
- 통합 답변 제공: 문서, 로그, 도메인 지식 기반 근거 포함 답변 (`SRS-012`)
- 도메인 기반 답변: 매뉴얼, SOP, 트러블슈팅 문서 기반 답변 (`SRS-017`)
- 체계적 답변 포맷: 사전 정의 템플릿 적용 (`SRS-018`)
- 관련 이미지 제공: 장비, 부품, 공정 이미지 조회 및 표시 (`SRS-019`)

### 1.5 자료 업로드 및 지식 저장소
- 파일 업로드: Drag & Drop, 파일 탐색기, 확장자/용량 검증 (`SRS-013`)
- 업로드 이력 기록: 메타데이터, 감사 추적, 12개월 이상 보존 (`SRS-014`)
- VectorDB 저장: 업로드 파일 벡터화, 메타데이터 저장, 답변 활용 (`SRS-015`)
- VectorDB 생성 및 관리: 목적별 DB 생성, 별칭 관리 (`SRS-016`)

### 1.6 음성 I/O
- 음성 질의: 마이크 입력, STT 변환, 질의 입력창 반영 (`SRS-020`)
- 음성 답변: TTS 변환 및 재생 (`SRS-021`)
- 발화 시점 호출: 핫워드 또는 버튼으로 음성 입력 시작 (`SRS-022`)
- 음성인식 팝업 호출: 특정 음성 발화에 따른 웹 사이트 열람 (`SRS-023`)
- 음성 응답 성능 보장 (`SRS-036`)

### 1.7 관리자 기능
- 관리자 페이지 접근 제어: 관리자와 일반 사용자 구분 (`SRS-024`)
- 사용자 및 이력 관리: 사용자 정보, 로그인/사용 이력, 파일 이력, 질의 이력 조회 (`SRS-024`)
- 사용자 서비스 이용 제지 (`SRS-024`)
- AI 모델 버전 관리: 등록, 조회, 변경, 이력 관리 (`SRS-025`)
- 파일 정책 관리: 업로드, 다운로드, 삭제 제한 정책 (`SRS-026`)
- 로그 관리: 로그인, 오류, 알람 등 주요 이벤트 로그 검색/필터 (`SRS-027`)

### 1.8 통계 분석 및 리포팅
- DB 기반 통계 분석: 기간, 변수, 조건별 통계값 산출 (`SRS-028`)
- 통계 시각화: 그래프 제공 가능 결과의 차트화 (`SRS-028`)
- 자연어 통계 분석: 채팅 질의 기반 DB 조회 및 분석 (`SRS-029`)
- 분석 결과 내보내기: PDF, CSV 등 파일 저장 (`SRS-030`)

### 1.9 데이터 라우팅
- 기본 데이터베이스 자동 선택: 질의 의도와 권한 기반 검색 대상 선택 (`SRS-031`)
- 선택 사유 제공: 자동 선택 DB 목록과 이유 표시 (`SRS-031`)
- 명시적 데이터 소스 범위 제한: 사용자가 지정한 데이터 소스 내 검색 (`SRS-032`)
- 출처 표시: 지정 데이터 소스 기반 답변의 근거 문서 표시 (`SRS-032`)

### 1.10 답변 검증 및 이력
- 피드백 저장: 좋아요/싫어요 버튼, 답변 품질 피드백 기록 (`SRS-033`)
- 질의 이력 조회: 기간 또는 키워드 검색 (`SRS-034`)
- 질의/응답 이력 삭제: 개별/다중 선택 소프트 삭제 (`SRS-034`)

### 1.11 운영, 배포, 백업
- 텍스트 질의 응답 성능: 10초 이내 응답, 60초 타임아웃 (`SRS-035`)
- 하이브리드 확장 아키텍처: 온프레미스 설치, 외부 모델/API 전환 가능 구조 (`SRS-037`)
- 백업 정책: 원본 문서, 벡터데이터, 메타데이터, 사용자 설정 일일/월간 백업 (`SRS-038`)
- 무중단 모델 교체: 롤링 배포 등으로 서비스 중단 없는 업데이트 (`SRS-039`)
