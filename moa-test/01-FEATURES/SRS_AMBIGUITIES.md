## 우선 보완 권고

1. 보안 정책값: 로그인 실패 횟수, 잠금 시간, 세션 만료 시간
2. 파일 정책값: 허용 확장자, 용량, 보존기간, 삭제 정책
3. 설비 제어 권한: 실행자/승인자/관리자 역할 분리
4. 데이터 소스 권한: VectorDB 소유자, 공유 범위, 출처 표시 기준
5. 운영 기준: 백업 스케줄, 모델 교체 롤백 조건, 외부 API 비용 통제
