## 9. 결함 판정

| 등급 | 기준 |
| --- | --- |
| Blocker | 인증 불가, 관리자 접근 통제 실패, 설비 제어 무권한 실행, 데이터 손실 |
| Critical | 성능 기준 미달, 백업/복구 실패, 모델 교체 중 서비스 중단, 감사 로그 누락 |
| Major | 주요 기능 실패, 파일 정책 미적용, 출처 표시 누락, 알람 통지 실패 |
| Minor | 메시지 문구 오류, 표시 순서 오류, 비핵심 필드 누락 |

## 10. 완료 조건

- VP-001 ~ VP-022 검증 항목이 모두 실행된다.
- P0 결함은 모두 해결되어야 한다.
- P1 결함은 릴리스 책임자 승인 없이 미해결 상태로 종료할 수 없다.
- 각 검증 항목은 실행 일시, 실행자, 환경, 결과, 증적 경로를 기록해야 한다.
